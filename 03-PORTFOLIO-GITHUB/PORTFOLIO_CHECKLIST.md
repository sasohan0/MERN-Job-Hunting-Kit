# Portfolio Website Checklist (MERN/React)

## Home page must answer 3 questions
- [ ] Who are you?
- [ ] What proof do you have?
- [ ] How can they contact you?

## Must-have sections
- [ ] Hero: target role + stack + 1 proof line
- [ ] Projects: 2–4 best projects (not 10)
- [ ] Project case studies: problem → solution → tools → results
- [ ] Skills: grouped (Frontend/Backend/Tools)
- [ ] About: short, professional (6–8 lines)
- [ ] Contact: email + LinkedIn + GitHub

## Portfolio quality checks
- [ ] Mobile responsive
- [ ] Fast load, no broken images
- [ ] Buttons work (live/code links)
- [ ] Each project has screenshots + features
