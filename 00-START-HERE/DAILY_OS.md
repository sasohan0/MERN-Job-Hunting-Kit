# Daily Job Hunt OS (15–60 Minutes)

This is the minimum system. Do it every day.

## A) 15-Minute Version (Busy day)
- [ ] Apply to 1 job (Direct/LinkedIn)
- [ ] Apply to 1 job (Facebook groups)
- [ ] Update tracker (2 minutes)
- [ ] One improvement (resume bullet / project README / LinkedIn keyword)

## B) 30-Minute Version (Normal day)
- [ ] Find 10 postings (save links)
- [ ] Apply to 3 postings
- [ ] Send 2 targeted DMs/emails
- [ ] Update tracker + follow-up list

## C) 60-Minute Version (High output day)
- [ ] Apply to 5–8 postings (mix channels)
- [ ] 5 targeted outreach messages
- [ ] One portfolio/GitHub improvement (README/screenshot/feature)
- [ ] 15 minutes interview practice (see Interview folder)

## Follow-up Rule (Simple)
- Day 0: Apply + DM/email
- Day 2: Follow-up 1
- Day 5: Follow-up 2 (final)
