# START HERE — How to Use This Repository

This repo is a **system**. Follow it like a playbook.

## 1) Your Non‑Negotiables (Set Today)
- [ ] Google Drive folder (Resume / Portfolio Assets / Screenshots / Certificates)
- [ ] Updated resume (PDF + DOCX)
- [ ] Portfolio live link
- [ ] GitHub updated (pinned projects + README)
- [ ] LinkedIn updated (headline, about, featured, skills)
- [ ] Application tracker ready (template in this repo)

## 2) The Job Hunting Ladder (Use All Three)
**Level 1 — Direct Apply:** Company career pages (high trust).  
**Level 2 — LinkedIn:** Apply + outreach (best ROI when consistent).  
**Level 3 — Facebook:** Fast + volume (apply quickly, but with proof).

## 3) Daily Rhythm (Simple)
- **15–20 min:** Find new posts (LinkedIn/FB/job boards)
- **20–40 min:** Apply (3+ high-quality applications)
- **10 min:** Follow-up + tracker update
- **10 min:** Improve assets (resume/portfolio/GitHub)

## 4) Submission Rules (Discord)
When you submit in **#task-submit**, always include:
- Resume link (latest version)
- Portfolio link
- GitHub link
- Post links you applied to (proof)
- Short reflection: what worked / what didn’t

Use the template here:
➡️ `00-START-HERE/TASK_SUBMISSION_TEMPLATE.md`
