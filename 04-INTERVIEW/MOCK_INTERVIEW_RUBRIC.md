# Mock Interview Script + Scoring Rubric

## How it works
Mentor asks → Student answers → Mentor scores (1–5)

### Q1: Tell me about yourself
**Mentor:** Tell me about yourself.  
**Student:** (Present → Past → Future)

Score on:
- Clarity (1–5)
- Proof (1–5)
- Relevance (1–5)
- Confidence (1–5)

### Q2: Biggest project challenge
**Mentor:** Tell me about a challenge you faced in a project.  
Use STAR.

Score on:
- Problem clarity
- Action ownership
- Result metrics
- Reflection

### Q3: Why this role?
**Mentor:** Why do you want this role/company?

Score on:
- Specificity
- Alignment with skills
- Motivation realism

### Q4: Technical mini question
**Mentor:** Explain how JWT works in a MERN app.

Score on:
- Correctness
- Structure
- Communication
